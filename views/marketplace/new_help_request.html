This page provides a form to submit an request for help on a project. You will only be able to submit a request for projects for which you are registered as a project coordinator. You should give a good description of the work you need doing and any ideas of how the work might fit into an undergraduate research project.

All requests are checked by an administrator for approval. You will get an email confirming that your request has been submitted and a follow up email when an admin decision has been made.
